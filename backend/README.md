# backendプロジェクトのREADME

## プロジェクトを作る

```
$ cd $HonoZodOpenApiSample
$ bun create hono@latest backend
```

## 外部依存パッケージのインストール

```
$ cd backend
$ bun install
$ bun install @hono/zod-openapi
$ bun install @hono/swagger-ui
```

## devサーバを起動する

To run:
```sh
bun run dev
```
open http://localhost:3000
